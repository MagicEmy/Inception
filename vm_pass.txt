database
docker ps
docker exec -it <container_id_or_name> bash
access the shell of the container:
	docker exec -it <container_id_or_name> bash
	mysql -u <MariaDB username> -p
	Enter the password and press Enter
	SHOW DATABASES;
e.g.
	docker exec -it my_mariadb_container bash
	mysql -u root -p
	SHOW DATABASES;

If you want to check the content of a specific database, 
you need to select it using the USE command. 
For example, to select a database named my_database,

USE my_database;
SHOW TABLES;
SELECT * FROM my_table;
